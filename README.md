# 마음톡 (Maeumtalk) — 첫 시청각 AAC

말이 아직 트이지 않은 **3–4세 아동을 위한 모바일 AAC(보완대체의사소통) 보드**입니다.
아이가 그림 카드를 누르면 한국어 음성으로 대신 말해 줍니다.

**라이브:** https://skdyddns-max.github.io/maeumtalk/

## 기능

- **4개 주제 보드** — 꼭 필요한 말 / 먹고 싶어요 / 하고 싶어요 / 내 마음 (각 4장)
- **항상 뜨는 필수 2장** — 그만해요 · 아파요
- **탭 → 음성** — 카드를 누르면 한국어 TTS(느린 속도)로 발화 + 진동 + 큰 선택 피드백
- **보호자 목소리 녹음** — 카드별로 엄마·아빠 목소리를 직접 녹음. 이 기기(브라우저 IndexedDB)에만 저장되어 안전하며, 녹음이 있으면 TTS 대신 그 목소리로 재생
- **오프라인 지원(PWA)** — 홈 화면에 설치 가능, 네트워크 없이도 동작

## 기술

순수 정적 사이트입니다. 서버·빌드 과정이 없습니다.

- `index.html` — 앱 전체 (마크업 + 스타일 + 바닐라 JS). React/서버 의존 없음
- `manifest.webmanifest`, `sw.js` — PWA (설치 · 오프라인 캐시)
- `aac-*.png` — 픽토그램(스프라이트 시트 및 개별 이미지)

원본은 vinext(Next.js RSC) 프로토타입이었고, GitHub Pages 배포를 위해 자체 완결형 정적 페이지로 이식했습니다.

## 수정 시 주의

앱을 수정해 다시 배포할 때는 `sw.js`의 `VERSION` 값을 반드시 올려야 사용자에게 새 버전이 반영됩니다.
